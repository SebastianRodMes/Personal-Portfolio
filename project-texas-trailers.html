Texas Trailers CR | Digital Ecosystem In Development

E-Commerce Architecture Web Development

A comprehensive B2C and B2B solution for trailer import and sales in Costa Rica. This project modernizes the operations of a leading import company, transforming a manual, decentralized sales process into a unified digital experience. Utilizing a BaaS (Backend-as-a-Service) pattern with Firebase, it resolving data fragmentation through a distributed hybrid architecture: a blazing-fast public React web app and a mobile app for admin field operations.

React 18 TypeScript Tailwind CSS Firebase Auth Firestore Firebase Storage

The Architecture Challenge

The business required two distinct applications to operate on the same data in real-time. Admins needed a mobile app to upload photos and update prices directly from the sales yard, while customers needed a fast, public-facing web catalog available 24/7. To bridge this gap, Google Firebase ecosystem was chosen as the central core connecting both the Web Client and the Android Native mobile app.

2

Business Impact: Centralizing Operations

Before this ecosystem, stock tracking was fragmented between the sales yard and varying spreadsheet records. Now, Inventory Centralization ensures customers only see what is truly available. Sales Optimization is achieved with advanced catalog filtering on the web front, drastically reducing the time customers spend inquiring about non-existent items.

3

Future Evolution

Currently, the platform fulfills Phase 1: a Read-Only Web application for customers and a Write-Heavy Native Android App for administrators. Phase 2 is actively in progress, integrating push notifications for special offers and an online web reservation system. Later, a full Web Dashboard migration is planned to replace manual workflows entirely.
